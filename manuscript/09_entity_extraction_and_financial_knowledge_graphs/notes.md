# Chapter 9 — Entity Extraction and Financial Knowledge Graphs — Research Notes

## Research Needed
- [ ] Financial NER benchmarks: FIN5, SEC-NER dataset, existing model performance baselines
- [ ] Ticker symbol disambiguation approaches (context-based, knowledge-base-linked)
- [ ] CUSIP, LEI, ISIN identifier systems and cross-referencing databases
- [ ] Relation extraction models for financial text: SpERT, REBEL, LLM-based extraction
- [ ] Knowledge graph construction tools: Neo4j, Amazon Neptune, RDF/OWL representations
- [ ] Temporal knowledge graph models: TComplEx, DE-SimplE, temporal graph networks
- [ ] Supply chain knowledge graph examples: FactSet supply chain data, NLP-extracted supply chains
- [ ] Graph centrality metrics as financial features: academic papers on network-based signals
- [ ] SEC filing parsing for entity and relation extraction: XBRL linkages
- [ ] Activist investor detection from filing language and ownership graph changes
- [ ] Community detection algorithms applicable to financial knowledge graphs
- [ ] Multi-modal graph enrichment: linking text-extracted relations with FactSet/Bloomberg structured data
