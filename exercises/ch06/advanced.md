# Chapter 6 — Earnings Calls and Corporate Filings: Advanced Exercises

## Exercise 6.6: Multi-Modal Earnings Analysis
Build a system that combines textual features from earnings calls with structured financial data (EPS surprise, revenue, guidance) to predict post-earnings drift. Evaluate the marginal contribution of text features using ablation analysis.

## Exercise 6.7: CEO Deception Detection
Using linguistic cues identified in academic literature (increased use of general terms, fewer self-references, more extreme positive language), build a classifier for detecting potentially deceptive language in earnings calls. Validate against subsequent restatements or enforcement actions.
