# Chapter 13 — Risk, Ethics, and Regulation: Intermediate Exercises

## Exercise 13.4: Model Explainability Report
Build an explainability layer for a financial sentiment classifier using SHAP or attention-based explanations. Generate a human-readable report for a compliance officer that explains why the model made a specific prediction.

## Exercise 13.5: Adversarial Robustness Testing
Test a financial NLP model's robustness against adversarial inputs: synonym substitution, character perturbation, and semantically equivalent paraphrases. Measure how much each attack type degrades classification accuracy.
