# Chapter 2 — Building Your Financial Text Dataset: Foundations Exercises

## Exercise 2.1: SEC EDGAR Scraper
Build a scraper that downloads the most recent 10-K filing for a given ticker using the SEC EDGAR full-text search API. Parse the HTML into clean plain text and extract the Risk Factors section.

## Exercise 2.2: News API Comparison
Using two different news APIs (e.g., NewsAPI and GDELT), fetch articles about the same company over a one-week window. Compare coverage, overlap, and metadata quality.

## Exercise 2.3: Data Quality Audit
Download 50 earnings call transcripts and compute basic quality metrics: average length, missing section rates, duplicate detection, and encoding error frequency.
