# Exercises — Applied NLP for Finance

Each chapter includes exercises at three difficulty levels:

## Tier System

### Foundations
Core implementation tasks that reinforce the chapter's key concepts. Every reader should complete these exercises. They typically involve implementing a specific technique covered in the chapter using the provided utilities in `src/`.

### Intermediate
Extensions and variations that deepen understanding and encourage experimentation. These exercises ask you to modify, extend, or combine techniques — for example, adapting a pipeline to a new data source or comparing two approaches on the same dataset.

### Advanced
Open-ended research problems suitable for capstone projects, independent study, or production prototyping. These exercises may require sourcing your own data, designing experiments, or building end-to-end systems. Some are deliberately underspecified to encourage creative problem-solving.

## Structure

Each chapter directory (`ch01/` through `ch13/`) contains:
- `foundations.md` — Foundations exercises
- `intermediate.md` — Intermediate exercises
- `advanced.md` — Advanced exercises

## Getting Started

1. Ensure you've installed the companion package: `pip install -e ".[notebooks]"`
2. Complete the corresponding chapter notebook before starting exercises
3. Start with Foundations and progress upward
4. Solutions are not provided intentionally — the learning is in the implementation
