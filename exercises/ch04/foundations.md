# Chapter 4 — Transformers and LLMs for Financial Text: Foundations Exercises

## Exercise 4.1: FinBERT Sentiment Pipeline
Build an end-to-end sentiment classification pipeline using FinBERT. Process a batch of 100 financial headlines, handle tokenization edge cases, and output a DataFrame with text, predicted label, and confidence score.

## Exercise 4.2: Prompt Engineering for Classification
Design three different prompt templates for financial sentiment classification using an LLM API. Test each on 50 labeled examples and compare accuracy, consistency, and cost.

## Exercise 4.3: Zero-Shot vs. Fine-Tuned Comparison
Compare zero-shot LLM classification against fine-tuned FinBERT on the same 200-sentence test set. Report accuracy, F1, latency, and cost per prediction.
