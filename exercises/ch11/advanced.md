# Chapter 11 — Backtesting NLP Strategies: Advanced Exercises

## Exercise 11.6: Multiple Testing Correction
Apply multiple testing correction methods (Bonferroni, Holm, and the deflated Sharpe ratio) to a set of 50 NLP signal variants. Compare how many signals survive each correction method and discuss the implications for signal discovery.

## Exercise 11.7: End-to-End Strategy Evaluation
Design and execute a complete strategy evaluation for a text-based trading signal. Include walk-forward backtesting, multiple benchmarks, transaction costs, capacity analysis, and regime-conditional performance breakdown. Write a strategy research note documenting your findings.
