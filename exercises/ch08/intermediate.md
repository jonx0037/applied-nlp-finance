# Chapter 8 — Social Media and Alternative Text: Intermediate Exercises

## Exercise 8.4: Attention vs. Sentiment Decomposition
Separate the attention component (volume of mentions) from the sentiment component (tone of mentions) for social media data. Test whether attention or sentiment has stronger predictive power for short-term returns and volatility.

## Exercise 8.5: Cross-Platform Signal Comparison
Collect data about the same set of stocks from two different social media platforms (e.g., Reddit and StockTwits). Compare signal characteristics including lead/lag relationships, noise levels, and coverage breadth.
