# Chapter 8 — Social Media and Alternative Text: Foundations Exercises

## Exercise 8.1: Social Media Noise Filter
Implement a preprocessing pipeline for financial social media posts that filters spam, removes bot-generated content, normalizes cashtags and hashtags, and handles common abbreviations used in trading communities.

## Exercise 8.2: Bot Detection Heuristics
Build a rule-based bot detection system using account-level features (post frequency, account age, follower ratio) and content-level features (template matching, repetition). Test it on a sample of posts from a financial subreddit.

## Exercise 8.3: WallStreetBets Sentiment Tracker
Build a daily sentiment tracker for a set of tickers mentioned on WallStreetBets. Aggregate post-level sentiment by ticker and day, and compare the resulting signal against actual stock returns over a one-month window.
