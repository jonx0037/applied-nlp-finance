# Chapter 8 — Social Media and Alternative Text: Advanced Exercises

## Exercise 8.6: Coordinated Manipulation Detection
Design a system to detect coordinated posting campaigns around specific stocks. Use network analysis of posting patterns, temporal clustering, and content similarity to identify suspicious coordinated activity.

## Exercise 8.7: Retail Flow Nowcasting
Build a model that uses social media text features to nowcast retail order flow. Evaluate against actual retail flow data (e.g., from public broker disclosures) and assess whether social media provides a leading signal.
