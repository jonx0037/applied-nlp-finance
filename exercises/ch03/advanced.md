# Chapter 3 — NLP Foundations for Finance: Advanced Exercises

## Exercise 3.6: Custom Financial Tokenizer
Design and implement a financial-domain tokenizer that correctly handles ticker symbols, monetary amounts, percentages, dates, and financial abbreviations. Benchmark it against standard tokenizers on a corpus of SEC filings.

## Exercise 3.7: Representation Evaluation Framework
Build a systematic evaluation framework that tests text representations on three downstream financial tasks (sentiment classification, topic clustering, and document retrieval). Rank at least four embedding methods and analyze where each excels or fails.
