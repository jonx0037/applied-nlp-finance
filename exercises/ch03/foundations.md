# Chapter 3 — NLP Foundations for Finance: Foundations Exercises

## Exercise 3.1: TF-IDF from Scratch
Implement TF-IDF vectorization from scratch (without scikit-learn) on a corpus of 100 financial news headlines. Compare your output against scikit-learn's TfidfVectorizer to verify correctness.

## Exercise 3.2: Embedding Comparison
Generate embeddings for 50 financial sentences using (a) Word2Vec averaging, (b) a pre-trained sentence transformer, and (c) a financial-domain model. Compare the three using cosine similarity on known-similar and known-dissimilar pairs.

## Exercise 3.3: Dictionary-Based Sentiment
Implement sentiment scoring using the Loughran-McDonald financial sentiment dictionary. Score 20 earnings press releases and compare the results against manually assigned sentiment labels.
