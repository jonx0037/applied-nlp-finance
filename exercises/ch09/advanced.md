# Chapter 9 — Entity Extraction and Knowledge Graphs: Advanced Exercises

## Exercise 9.6: Graph-Based Risk Propagation
Build a contagion model on a financial knowledge graph that simulates how a negative shock to one company propagates through supply chain and ownership relationships. Calibrate propagation parameters using historical event data.

## Exercise 9.7: Knowledge-Enhanced NLP
Design a system that uses knowledge graph context to improve NLP task performance. For example, use entity relationship information to disambiguate sentiment targets or to enrich document representations for classification.
