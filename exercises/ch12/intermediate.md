# Chapter 12 — Deployment and Production Systems: Intermediate Exercises

## Exercise 12.4: Streaming Pipeline
Build a streaming text processing pipeline that ingests news articles in real time, computes sentiment scores, and updates a live signal dashboard. Handle backpressure, deduplication, and late-arriving data.

## Exercise 12.5: A/B Testing Framework
Design an A/B testing framework for comparing two NLP model versions in production. Implement traffic splitting, metric collection, and a statistical test for determining when one model significantly outperforms the other.
