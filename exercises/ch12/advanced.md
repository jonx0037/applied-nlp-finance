# Chapter 12 — Deployment and Production Systems: Advanced Exercises

## Exercise 12.6: Production Architecture Design
Design a complete production architecture for a multi-model NLP signal generation system. Include data ingestion, model serving, signal computation, monitoring, and alerting components. Document scaling considerations and failure modes.

## Exercise 12.7: RAG for Financial Research
Build a Retrieval-Augmented Generation system for financial research. Index a corpus of earnings call transcripts and SEC filings, implement hybrid search (semantic + keyword), and build an interface that answers financial questions with cited sources.
