# Data Acquisition Guide

This directory contains data for the companion notebooks. Due to size and licensing constraints, most raw data is not included in the repository and must be acquired separately.

## Directory Structure

- `sample/` — Small, redistributable sample datasets for immediate use
- `raw/` — Raw data (gitignored — acquired by reader)
- `processed/` — Processed data (gitignored — generated by notebooks)

## Acquisition Instructions

### SEC EDGAR (Chapters 2, 6, 9)

```bash
# Using sec-edgar-downloader
from sec_edgar_downloader import Downloader
dl = Downloader("YourName", "your@email.com", "data/raw/edgar")
dl.get("10-K", "AAPL", after="2020-01-01")
```

### Financial News (Chapter 2)

Requires a NewsAPI key. Set `NEWSAPI_KEY` in your `.env` file.

### Reddit Data (Chapter 8)

Requires Reddit API credentials. Set `REDDIT_CLIENT_ID`, `REDDIT_CLIENT_SECRET`, and `REDDIT_USER_AGENT` in your `.env` file.

### FOMC Data (Chapter 7)

Available directly from the Federal Reserve website. The scraper in `src/data/scrapers.py` automates retrieval.

## Reproducibility

For exact reproducibility of results in the book, we recommend snapshotting your `data/raw/` directory after initial acquisition. Options:
- **Manual:** Archive `data/raw/` with a date-stamped filename
- **DVC:** Use Data Version Control for systematic dataset versioning
- **git-lfs:** For datasets under 2 GB

See the book's Chapter 2 for detailed discussion of data versioning strategies.
